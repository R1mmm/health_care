## 주요 기능

### 알림 서비스 💡
    - 본인이 예약한 운동시간
    - 친구가 운동했을 시(친구 추가 기능)
    - 최근 접속 기간이 오래됐을 때
    - 알림 메일, 알림창
### 캘린더 기능 📅
    - 그룹 공유 캘린더 → 출석체크 가능
    - 개인 캘린더
    - 캘린더 댓글(추후에 다시 결정)
### 친구 기능 👨‍👩‍👧‍👦
    - 친구 추천, 목록, 추가
    - 그룹 생성
